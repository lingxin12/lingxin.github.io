购买vps

本例使用阿里云购买，由于是实验，所以找的比较稳定的服务器，但阿里云收费比较高，建议用阿里云购买服务器后上国外的vps网站购买，流量计费的都比较便宜。

（这里用阿里云演示，以后可能会有别的vps购买流程）

具体内容见附录1

这个就是创建的实例，需要记住ip地址，登入用户名及密码。

ping一下ip地址发现能够ping通则连接正常。

域名解析

注册一个域名，点击解析。（这里用华为云做示例）

管理解析。

在下面两行把ip改成vps的ip地址。

然后ping一下ip地址，能够ping成功即可。

远程连接配置服务端

打开Xshell6并新建会话。

输入主机ip地址并使用默认端口22。

点击隧道，取消“转发X11连接到”。

输入用户名。

输入密码。

如图显示，连接成功。

输入如下代码，下载v2ray一键安装脚本。

bash <(curl -s -L https://git.io/v2ray.sh)

（ 如果提示 curl: command not found ，那是因为你的 VPS 没装 Curl ubuntu/debian 系统安装 Curl 方法: apt-get update -y && apt-get install curl -y centos 系统安装 Curl 方法: yum update -y && yum install curl -y 安装好 curl 之后就能安装脚本了 ）

回车，回车，回车。

配置Shadowsocks，一路回车（可以不安装）。

确认安装信息。

安装完成。

开放浏览器端口

如果使用阿里云等国内的一些服务器，需要开放浏览器端口（不开的话没法没法联网，这就是为什么v2rayN明明连上了却上不了网的原因）。国外的服务器一般不需要这一步。

打开实例后，选择安全组，点击配置规则。

点击手动添加。

按步骤开放如上两个端口。

配置v2rayN

打开v2rayN-core，点击左上角服务器-编辑或添加[VMess]服务器。

然后输入地址，服务端上的端口，用户ID，并输入伪装的域名。

添加成功后可以右键测试一下ping值。

选择设置-路由设置，点击基础功能并一键配置基础功能。

点击启用路由高级功能，点击绕过大陆，并点击确定。
